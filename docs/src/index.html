HGR   Tool

HGR Tool is an open-source application for working with Apple II graphics files. It's implemented as a single-page web application, usable in any modern browser (Chrome is recommended). This is currently an early proof of concept, not a highly-polished product.

NOTE: the tool can make use of some newer File System Access API calls when they are available. Some conveniences, such as saving updates to an image without having to use the file download mechanism, are not available in the browser you're currently using. For a smoother experience, use Chrome, Edge, or Opera.

==> Run Hi-Res Image Editor <==

The tool works like other paint programs, but it's often wise to skim the documentation.

Don't have any images handy? Download samples.

The source code is available on github.
